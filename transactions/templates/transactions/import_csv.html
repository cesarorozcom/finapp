CSV Format Guidelines

Your CSV file should contain columns for transaction data. The system will automatically map columns based on the names you specify above.

Example CSV format:
date,description,amount,category
2023-01-15,Grocery Store,45.67,Food
2023-01-16,Restaurant,-25.50,Dining
2023-01-17,Gas Station,35.00,Transport
Tips:
  • Use positive amounts for expenses, negative for income
  • Category column is optional - leave empty for auto-categorization
  • Date format should be YYYY-MM-DD
  • Make sure your CSV has headers matching the column names above
